# 如何用codex建立个人知识库 第二大脑

## 来源

- 标题：如何用codex建立个人知识库 第二大脑
- 作者/公众号：白无常C4D
- 发布时间：2026-07-02 21:42 UTC+8
- 链接：https://mp.weixin.qq.com/s/wncxlGaNEG_TS4-qufCXHg

## 摘要

文章主张把个人知识库从“收藏资料”升级为“AI 可调用的长期上下文”。作者借鉴 Tiago Forte 的 Second Brain 和 Karpathy 的 LLM Wiki 思路，用本地 Markdown 目录、Agent 维护规则和 HTML 仪表盘，把灵感、来源、任务、日程、项目、领域记录组织起来。

## 可行动启发

- 用开放格式保存长期内容，避免被单一工具锁定。
- 把原始资料、结构化 Wiki 和维护规则分层管理。
- 让 Agent 按规则归档资料、任务、日程和项目。
- 用仪表盘把 Markdown 内容转成搜索、近期计划、日历和关系视图。
